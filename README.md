# dotfiles

A lightweight, fast shell environment optimized for Node.js / JavaScript development.

## Goals
- Fast shell startup
- Minimal abstractions
- Node-aware tooling
- Easy to understand and remove

## Includes
- zsh (no oh-my-zsh)
- starship prompt
- fnm for Node version management
- fzf + jq helpers
- git-delta
- direnv for per-project env vars

## Installation

```sh
git clone https://github.com/yourusername/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install.sh
